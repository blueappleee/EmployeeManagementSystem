To run the program you can run python3 login.py in the directory it is installed so ~/cs4471program/EMS/src/login_cmdUI/
When running the program you have 9 employees, 000001, 000002 for admin, 000003, 000004, 000005 for manager and the rest 00000[6-9] are regular employees
The current default password for all of them is just password
Once logged in you will go to the panel for the type of employee you logged in as and from there it will display options and prompt for input as needed
